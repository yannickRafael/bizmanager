### Dashboard (Início)
- **Menu Lateral**: Access Settings and Navigation.
- **Resumo**: View Total Revenue, Total Orders, and Pending Orders.
- **Ações Rápidas**: Quickly navigate to "Gerir Clientes" or "Gerir Pedidos".

### Client Management (Clientes)
- View a list of all registered clients.
- **Novo Cliente**: Register a new client with Name, Phone, and Address.
- **Detalhes do Cliente**: Tap any client to see:
    - **Dívida Total**: Sum of all unpaid amounts.
    - **Notas**: Add or edit notes about the client.
    - **Histórico**: List of all orders for this client.

### Order Management (Pedidos)
- View a list of orders sorted by date.
- **Novo Pedido**: Create an order for a new or existing client.
    - Select Client or Import from Contacts.
    - Choose Product (Frango/Ovos) and Quantity.
    - Price is auto-calculated.
- **Pagamentos**: Tap an order to register a payment.
    - Enter `Amount Paid`.
    - Status updates to `PARCIAL` or `PAGO` automatically.

### Settings (Definições)
- **Idioma**: Currently set to "Português".
